Algoritmo Arena Azul

Entradas:  
    reservas_previstas, datos_huesped, usuario_admin, clave_admin

Inicio
       Planeación de la demanda
    leer reservas_previstas
    calcular disponibilidad_habitaciones, disponibilidad_servicios
    guardar proyección

      Ingreso y registro de huéspedes
    leer datos_huesped
    validar nombre, apellido, documento
    si datos válidos entonces
        asignar habitación
        registrar check-in con fecha y hora
    sino
        escribir "Datos inválidos, por favor revisa tu registro"
    fin si

       Cálculo de ingresos y balance
    obtener ventas, costos
    ingresos_totales = sumar ventas
    costos_totales = sumar costos
    ganancia = ingresos_totales - costos_totales
    guardar reporte_diario

       Módulo administrador
    leer usuario_admin, clave_admin
    si credenciales correctas entonces
        mostrar total_huespedes
        mostrar huespedes_por_tipo
        mostrar total_mascotas
        mostrar disponibilidad_habitaciones
        mostrar disponibilidad_servicios
        mostrar ingresos_totales, costos_totales, ganancia
        si exportar == verdadero entonces
            generar archivo CSV
        fin si
    sino
        escribir "Acceso denegado"
    fin si

Fin
